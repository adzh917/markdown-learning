# Markdown 基础语法

## 1. 标题

标题用 `#` 表示，`#` 越多，标题级别越低。

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 2. 段落

普通文字直接写即可。两个段落之间空一行。

这是第一段。

这是第二段。

## 3. 加粗和斜体

```markdown
**这是加粗**
*这是斜体*
***这是加粗加斜体***
```

效果：

**这是加粗**

*这是斜体*

***这是加粗加斜体***

## 4. 列表

无序列表：

```markdown
- 学习标题
- 学习列表
- 学习代码块
```

有序列表：

```markdown
1. 打开编辑器
2. 创建 Markdown 文件
3. 写下今天学到的内容
```

任务清单：

```markdown
- [x] 学会标题
- [ ] 学会表格
- [ ] 学会插入图片
```

## 5. 引用

引用适合记录书摘、重点提醒或自己的思考。

```markdown
> 学习笔记的重点不是写得多，而是以后能看懂。
```

## 6. 代码

行内代码用反引号包起来，例如：`Ctrl + S`。

多行代码使用代码块：

```python
print("Hello, Markdown")
```

## 7. 链接

```markdown
[GitHub](https://github.com)
```

效果：

[GitHub](https://github.com)

## 8. 表格

```markdown
| 语法 | 用途 |
|---|---|
| `#` | 标题 |
| `-` | 列表 |
| `**文字**` | 加粗 |
```

效果：

| 语法 | 用途 |
|---|---|
| `#` | 标题 |
| `-` | 列表 |
| `**文字**` | 加粗 |
