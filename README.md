# Markdown Learning

这是我的 Markdown 学习仓库，用来记录 Markdown 语法、练习内容和学习笔记。

## 学习路线

1. 基础排版：标题、段落、加粗、分割线
2. 列表结构：无序列表、有序列表、任务清单
3. 内容整理：引用、代码、链接、图片
4. 表格表达：用表格记录概念、对比和计划
5. 进阶能力：数学公式、流程图、目录和笔记管理

## 目录说明

| 目录 | 用途 |
|---|---|
| `notes/` | 正式学习笔记 |
| `practice/` | Markdown 语法练习 |
| `templates/` | 常用笔记模板 |
